



## Deploy


```bash
solana program deploy ./target/deploy/hello_world.so
```

## Command

- Start Local Validator

```bash
solana-test-validator
```

- check config

```bash
solana config get
```

- airdrop

```bash
solana airdrop 5
```

- check balance

```bash
solana balance
````
